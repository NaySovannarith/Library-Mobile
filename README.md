# Library App

A Flutter-based library management app with modules for authentication, home, history, profile, scanning, and staff features.

## Features
- User authentication (login/signup)
- Home dashboard
- History tracking
- Profile management
- QR/Barcode scanning
- Staff module and roles

## Project Structure
- lib/
	- main.dart
	- auth/
	- config/
	- module/
	- staff/

## Requirements
- Flutter SDK (stable)
- Dart SDK (bundled with Flutter)
- Xcode (for iOS) and/or Android Studio (for Android)

## Setup
1. Install Flutter: https://docs.flutter.dev/get-started/install
2. Fetch dependencies:
	 - flutter pub get

## Run
- flutter run

## Build
- Android: flutter build apk
- iOS: flutter build ios
- Web: flutter build web

## Configuration
API configuration is located in lib/config/api_config.dart.

## Testing
- flutter test
